Problem A: Clock

Problem

がっちょ君はお気に入りの時計を持っている。ある日時計の長針が取れてしまい、どこかへなくしてしまった。しかし、がっちょ君はその時計を使い続けたいため短針だけで時刻を読み取りたいと考えている。

短針の情報θに対する時刻(時h、分m)を出力せよ。時計はいわゆるアナログ時計で、1から12の数が等間隔に時計回りに昇順に並んでいるものである。

Input

入力は以下の形式で与えられる。

θ

1行に短針の角度 θを表す整数が度数法(degree)で与えられる。 短針が指す方向は、12時を指す方向を0度としたとき、時計回りにθ度回転した方向である。

Constraints

  • 0 ≤ θ ≤ 359

Output

短針が与えられた角度をとる時の時刻を
h m
の形で1行に出力する。

時刻に午前午後の区別はないため0 ≤ h ≤ 11とする。

Sample Input 1

0

Sample Output 1

0 0

Sample Input 2

45

Sample Output 2

1 30

Sample Input 3

100

Sample Output 3

3 20
