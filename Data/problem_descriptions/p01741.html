マンハッタンでは，道路がx 座標またはy 座標が整数のところに通っている．すぬけ君の家とすめけ君の家はどちらも道路上にあり，直線距離(ユークリッド距離) はちょうど d である．すぬけ君の家からすめけ君の家まで道路に沿って移動するときの最短距離として考えられる最大値を求めよ．

Constraints

  • 0 < d ≤ 10
  • d は小数点以下ちょうど三桁まで与えられる

Input

d

Output

答えを一行に出力せよ．絶対誤差または相対誤差が 10−9 以下のとき正答と判定される．

Sample Input 1

1.000

Sample Output 1

2.000000000000

Sample Input 2

2.345

Sample Output 2

3.316330803765
