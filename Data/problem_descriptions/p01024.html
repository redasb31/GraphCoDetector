Problem I: Sum of Last Digits

Problem

3つの整数 n, m, k が与えられるので

n1%10  +  n1+m%10  +  n1+2m%10  + ... +  n1+(k-1)m%10

を計算せよ。 a % b は a を b で割ったときの余りを表す。

Input

n m k

1行に n, m, k が与えられる。

Constraints

入力は以下の条件を満たす
  • 0 ≤ n ≤ 1018
  • 0 ≤ m ≤ 109
  • 1 ≤ k ≤ 109

Output

答えを1行に出力せよ。

Sample Input 1

1 1 9

Sample Output 1

9

Sample Input 2

2 1 3

Sample Output 2

14

Sample Input 3

6 11 11

Sample Output 3

66

Sample Input 4

100 7 12

Sample Output 4

0

Sample Input 5

123 123 3

Sample Output 5

11
