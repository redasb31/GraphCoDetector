Problem F: Remainder Zero

Problem

整数の集合A,Bについて,次の条件を満たす整数 xがいくつ存在するか答えよ。

Ai mod x = 0 かつ x mod Bj = 0 がすべての i (1 ≤ i ≤ N), j (1 ≤ j ≤ M) について成り立つ。

(a mod b はaをbで割ったときの余りを意味する)

Input

N M
A1 A2 ... AN
B1 B2 ... BM

Constraints

入力は以下の条件を満たす。

  • 1 ≤ N, M ≤ 5
  • 1 ≤ Ai, Bj ≤ 1014 (1 ≤ i ≤ N, 1 ≤ j ≤ M)

Output

条件を満たす整数 x がいくつ存在するか一行で出力せよ。

Sample Input 1

1 2
18
6 9

Sample Output 1

1

Sample Input 2

1 2
256
2 4

Sample Output 2

7
